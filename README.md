# Portfolio Tattoo

This is a portfolio website for a tattoo artist.

## Setup

1. Clone the repository.
2. Run `npm install` in the root and in `portfolio-website`.
3. Start the development server with `npm run dev`.

## Structure

- `portfolio-website/` - Main Next.js app
- `src/` - Source files
- `public/` - Static assets
